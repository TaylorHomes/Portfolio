Decreedly

A functional and robust blog.

I was on a team with three other developers and we built this application in one week. Modeled after twitter, our goal was to create a highly functional and powerful application. Built with Python and Django, this application utilizes Django's REST Framework with a responsive API that communicates to a front-end built with Vue.js. We had never used Vue prior to this project, so there was a good bit of learning involved. We chose Vue because as a framework, it seemed to be versatile enough to meet our needs.

We developed a complex model system through Django that would allow users many different use cases. From viewing active followers, to blocking followers, we attempted to run through every instance in order to make our application highly functional.

We loaded fake data in order to test all of our components and were ultimately very satisfies with the results of our collective efforts. This was no doubt the fastest life cycle I've ever worked on, and it required a lot from our team, but I think it was worth it in the end. Below are some screenshots of the project.
